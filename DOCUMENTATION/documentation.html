Directory Structure

    HRM
    ├── app/  ->It contains all the backend code of our web application like Controllers,Providers,Middlewares etc.
    ├── bootstrap/ ->This directory contains app.php from where the whole framework bootstraps.
    ├── config/ ->This directory contains all the configuration files related to database, mail, session, services etc.
    ├── database/ ->This directory contains database migrations, model factories and seeds.
    ├── Modules/ ->This directory will contain all Modules which is used to build appplication    
    ├   ├─ Attendances/ -> This module is reponsible for managing attedance related work, and it will contain below folders.
    ├      ├─ Config/
    ├      ├─ Console/
    ├      ├─ Database/
    ├      ├─ Entities/
    ├      ├─ Http/
    ├      ├─ Providers/  
    ├      ├─ Resources/  
    ├      ├─ Routes/     
    ├      ├─ Tests/     
    ├      ├─ vendor/       
    ├   ├─ Configurations/ ->This module is reponsible for managing application settings related work
    ├   ├─ Dashboard/ ->This module is reponsible for managing dashboard related work
    ├   ├─ Employees/ ->This module is reponsible for managing employee related work
    ├   ├─ Holidays/ ->This module is reponsible for managing holidays related work
    ├   ├─ Payroll/ ->This module is reponsible for managing payroll and salary slip related work
    ├   ├─ Profile/ ->This module is reponsible for managing profile related work
    ├   ├─ Projects/ ->This module is reponsible for managing project related work
    ├   ├─ Role/ ->This module is reponsible for managing role related work
    ├   ├─ Tasks/  ->This module is reponsible for managing task and mytask related work
    ├   ├─ Vacations/ ->This module is reponsible for managing vacation and allocate leave related work
    ├── public/ ->This directory contains assets used in the application like images, javaScript and css.
    ├── resources/ ->This directory contains the frontend of the application.
    ├── routes/ ->This directory contains all the route definitions of the application.
    ├── storage/ ->This directory contains the compiled Blade templates,file caches and other files generated by framework.
    ├── tests/ ->This directory contains all of our automated tests file.
    ├── vendor/ ->This directory contains all the dependencies downloaded through Composer needed by our framework.

Requirments

  • PHP Version greater than or equal to 7.3
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • OpenSSL PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • PDO PHP Extension
  • Node Version v14.15.5(for compling assets)
  • Xampp or Wampp if you are installing your local system

Installation

Step#1 UnZip the Script

Step#2 Go to your .env file and set your database, database username and database password

Step#3 Open terminal from your project root folder

Step#4 Run php artisan migrate

Step#5 Run php artisan db:seed

Step#6 Run php artisan serve

Step#7 Installation Completed!.
